Hexo 个人博客搭建

发表于 2017-03-20 | 分类于 技术 |
字数统计

花了几个小时使用 Hexo 把个人博客搭建完成。 虽说整个过程不是非常复杂，但是还是需要仔细的阅读文档，逐步的将自己所需要的功能添加上去。一般需要添加的功能是个性主题和评论模块，代码部署在 Coding , 后面也可以考虑部署在 Github。

步骤如下：

  1. 安装 Git 工具，Nodejs（国内可能需要使用 淘宝镜像来加速下载包的速度）。
  2. 安装 hexo-cli， 文档详见 hexo
  1. 安装主题

参考文档 NexT文档， 按照文档配置好所需要的一切，然后生成，预览，部署。

不明白的地方多在网上搜一下，很多文章都写的很好。补充一下域名的配置，这里是购买的阿里云的域名。要注意的地方是，如果使用的是 coding 的 Pages 服务。域名解析需要配置成如下方式：

coding Pages服务 里面绑定好域名， 开启强制 HTTPS 访问，并申请 SSL/TLS 证书。

Done!
